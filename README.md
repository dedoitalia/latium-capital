# Latium Capital — website

Sito vetrina bilingue (IT/EN) per Latium Capital Ltd (London): investimenti in crediti commerciali verso la Pubblica Amministrazione italiana.

## Pagine
- `index.html` — Home
- `come-funziona.html` — Come funziona / How it works
- `investitori.html` — Investitori / Investors
- `cedenti.html` — Cedi le fatture / Sell invoices
- `faq.html` — FAQ
- `chi-siamo.html` — Chi siamo / About
- `contatti.html` — Contatti / Contact
- `assets/style.css`, `assets/site.js` — stile e toggle lingua

## Sito statico
Nessun build step. La root contiene `index.html`. Su Render: Static Site, publish path `./`.

> Avvertenza: sito a fini informativi. Capitale a rischio; rendimenti non garantiti. Testi da validare legalmente prima della pubblicazione (financial promotions FCA/FSMA).
